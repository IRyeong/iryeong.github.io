[경험 3 : Bucket Splash - 버킷 리스트 SNS 개발] - 최우수상 수상

  1. 개발 기간 : 2022.11 - 2022.11 (총 2주)
  1. 개발 인원 : 2명
  1. 개발 환경
    • Spring Boot를 사용하여 Java8로 구현
    • MySQL과 MyBatis 사용
    • Vue.js 사용
    • azure에 db 배포
  1. 프로젝트 개요

    사용자의 버킷 리스트를 관리하는 웹 사이트로,

    자신이 작성한 버킷 리스트를 다른 사용자들과 공유하며 소통하는 장입니다.

    회원가입과 로그인은 물론, 다른 사용자들을 팔로우할 수 있어 보다 연결된 느낌을 주며,

    자유 게시판에 버킷 리스트와 관련된 글을 쓰고 다른 사용자가 쓴 글에 좋아요나 댓글로 반응할 수 있습니다.

    또한 특정 버킷 리스트를 달성하기 위해 여러 사용자를 모집하는 모집 게시판도 이용할 수 있습니다.

  1. 맡은 역할
    • ERD 설계
    • jwt를 이용한 로그인과 로그아웃과 sha-256을 이용한 비밀번호 암호화
    • 버킷리스트, 자유 게시판, 모집 게시판, 팔로우, 좋아요 등 모든 CRUD api 개발 azure에 db 배포
